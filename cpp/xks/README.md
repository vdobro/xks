# XKS
XKS is a helper for memorization of large amounts of structured data like lists, tables, and directed graphs.
## Build
### Ubuntu dependencies
- g++ (with C++17 support)
- make
- cmake (>= 3.15)
- pkg-config
- libsigc++-2.0-dev
- libglibmm-2.4-dev
- libgtk-3-dev
- libcairomm-1.0-dev
- libatkmm-1.6-dev
- libgtkmm-3.0-dev
